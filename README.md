# Homebrew Tap for kloudlabs-io

Official Homebrew tap for tools from [kloudlabs-io](https://github.com/kloudlabs-io).

## Installation

```bash
brew tap kloudlabs-io/tap
```

## Available Formulas

### mkrel

Release management CLI with Git Flow and CalVer/SemVer support.

```bash
brew install kloudlabs-io/tap/mkrel
```

For more information, see the [mkrel repository](https://github.com/kloudlabs-io/mkrel).
